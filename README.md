# Trovia – Trek Detail & Landing Page

Internship task submission for Trovia's Web Development Intern role.

## Live Demo
[trovia-assignment.vercel.app](https://trovia-assignment.vercel.app)

## What's Built
- **Landing Page** – Hero, featured treks, how it works, adventure CTA, community, reviews, newsletter, footer
- **Trek Detail Page** – Sticky booking bar, interactive itinerary accordion, live seat count, price calculator, gallery, FAQ

## Tech Stack
- React (Vite)
- Inline styles + CSS animations (no external UI library)
- Deployed on Vercel

## Key Design Decisions
- Shifted palette from orange/red to forest green + gold — feels more authentic to the Himalayas
- Every section is conversion-focused: sticky CTA, urgency indicators, trust signals near the booking button
- Fully responsive — mobile hamburger nav, stacked layouts, bottom nav bar on small screens
- Scroll-reveal animations using IntersectionObserver, no heavy animation libraries

## Run Locally
```bash
npm install
npm run dev
```
